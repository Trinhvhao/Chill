# Hướng Dẫn Chạy Ứng Dụng heart-chill

heart-chill là ứng dụng web được xây dựng bằng **React** với **TypeScript** (các file `.tsx`), chạy trên nền **Node.js** và quản lý thư viện bằng **npm**.

---

## Công nghệ sử dụng

- **React**: Thư viện xây dựng giao diện người dùng hiện đại.
- **TypeScript (.tsx)**: Phát triển mã nguồn an toàn, tăng tính kiểm soát khi làm việc với React.
- **Node.js**: Nền tảng để chạy ứng dụng và các lệnh phát triển.
- **npm**: Quản lý cài đặt thư viện/phụ thuộc cho dự án.
---

## Hướng dẫn cài đặt & chạy ứng dụng

### 1. Cài đặt Node.js

Tải và cài đặt Node.js tại [https://nodejs.org/](https://nodejs.org/)

### 2. Clone mã nguồn về máy

```bash
git clone https://github.com/Trinhvhao/Chill.git
```

### 3. Chọn đúng folder

```bash
cd Chill
cd heart-chill
```

### 4. Cài đặt thư viện (dependencies)

```bash
npm install
```


### 5. Chạy ứng dụng React ở chế độ phát triển

```bash
npm run dev
```

Thông thường, ứng dụng sẽ khởi chạy tại:  
[http://localhost:3000](http://localhost:3000)  
(Hoặc đường link sẽ hiện ra trong dòng lệnh khi chạy thành công.)
